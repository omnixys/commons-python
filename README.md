# omnixys-commons

Omnixys shared platform contracts — transport-neutral DTOs, enums, errors, and models.

## Installation

```bash
pip install omnixys-commons
```

## Features

- **DTOs**: Data Transfer Objects for API communication
- **Enums**: Shared enumerations across services
- **Errors**: Standardized error handling with error codes
- **Models**: Pydantic models for domain contracts

## Usage

```python
from commons import dtos, enums, errors, models
```
